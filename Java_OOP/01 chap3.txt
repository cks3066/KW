상수

자바의 일반적인 상수

변수 - 변하는 수;

변수의 이것을 상수로 하고싶으면 final 이라고 하면 된다. final > [수정 불가, 오버라이딩 금지시에도 사용] 

const가 아니고?

java에서는 변수를 상수로 변경하고 싶으면 final을 붙여주면 된다.

키워드 final이 붙어있는 변수

final int MAX_SIZE = 100;
// 상수 선언의 예 초기화 하지 않으면 딱 한번 초기화 가능하다!

변수라는 것은 로컬 변수가 이름을 어떻게 짖든 상관없지만
상수는 글로벌 상수로써 모두가 사용하기 때문에 모든 문자를 대문자로 해야된다.

상수는 대문자로 설정한다!!!

지금보면 변수에 대해서 final 이라고 이름을 붙였어요 이렇게 되면 고정의 값을 가지게 되는거죠

그러면 값은 있어야 되는 거잖아요!

final int MX_SIZE = 100;
final char CONST_CHAR - '상';

리터럴에 대한 이해

	리터럴
		자료형을 기반으로 표현되는 상수를 의미한다.
		ex) int num1 = 5 + 7;
		ex) double num2 = 3.3 + 4.5;
		정수는 무조건 int형으로 인식하기로 약속되어 있음
		따라서 5와 7은 정수형 리터럴이다.
		그리고 3.3과 4.5는 실수형 리터럴이다.
		리터럴이라는 표현은 상수라는 표현으로 대신하는 경우가 많다.

		메모리 상에 들어가는 리터럴, 상수 -> 연산, 출력용

정수형 상수의 표현 방법
	int num1 = 123;
	int num2 = 0123;
	int num3 = 0x123;

	System.out.println("num1: " + num1);

long형 상수(리터럴)의 표현 방법
	System.out.println("3147483647 + 31478=483648)'
	컴파일시 인티져 넘버 투 롱 라는 오류 메시지를 전달한다
	System.out.println("3147483647L+ 31478=483648L)'

	숫자 뒤에 L을 붙여주면 long 형 상수가 된다

정수형 상수의 이진수 표현방법과 언더바 삽입
byte seven = 0B111;
	0B, 0b를 붙여서 이진수 표현 0b, 0, 0x 이진수 팔진수 십윤진수

int num = 100_000_000;
int num = 12_34_56_78_90;

실수형 상수(리터럴)
	System.out.printfln(3.0004999+2.0004999);
	실수는 기본 double 형 double 형임을 명시하기 위해 d또는 D삽입

뒤에 D L F 등등 리터럴의 형을 정할 수 있다.

실수형 상수의 e표기법
	3.4e3 > 3.4x10^3 = 3400.0
	3.4e-3 > 3.4x10^-3 = 0.0034

부울형 상수와 문자형 상수
	true false
	부울형 상수
	'한' '글' 'A' 'Z'
	
이스케이프 시퀀스(escape sequences)
	\n 백스페이스
	\t 탭
	\\ 백슬래시
	\' 작은따옴표
	\" 큰따옴표
	\n 개 행문자
	\r 캐리지 리턴 문자
	화면상의 어떠한 상황 또는 상태를 표현하기 위해 약속된 문자
